## Install

With [fisherman]

```fish
fisher sol
```

## Features

* Git
    * Clean
    * Dirty / Touched
    * Staged
    * Staged + Dirty
    * Stashed
    * Unpushed commits (ahead)
    * Unpulled commits (behind)
    * Unpulled and unpushed commits (diverged)
    * Detached HEAD
    * Branch name
* $status
* $CMD_DURATION
* $status in context
* Background jobs
* Superpowers (sudo)
* Host information
